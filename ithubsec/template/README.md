# Шаблоны документов

В этой папке лежат .docx-шаблоны, из которых система генерирует документы при создании заявки.

## Файлы

| Тип заявки (код) | Файл шаблона |
|------------------|--------------|
| Справка об обучении | `Шаблон_справка_об_обучении.docx` |
| Справка для военкомата | `Шаблон_справка_военкомат.docx` |
| Справка рекомендован к зачислению | `Шаблон_справка_рекомендован_к_зачислению_.docx` |

## Плейсхолдеры

В тексте шаблона можно использовать подстановки в формате `{{ИМЯ}}`. Они заменяются при генерации:

| Плейсхолдер | Значение |
|-------------|----------|
| `{{FULL_NAME}}` | ФИО (Фамилия Имя Отчество) |
| `{{FIRST_NAME}}` | Имя |
| `{{LAST_NAME}}` | Фамилия |
| `{{PATRONYMIC}}` | Отчество |
| `{{GROUP_NAME}}` | Группа |
| `{{EMAIL}}` | Email |
| `{{TICKET_TITLE}}` | Заголовок заявки |
| `{{TICKET_DESCRIPTION}}` | Описание заявки |
| `{{TICKET_ID}}` | Номер заявки |
| `{{CREATED_DATE}}` | Дата создания заявки (дд.мм.гггг) |
| `{{CREATED_TIME}}` | Время создания |
| `{{CURRENT_DATE}}` | Текущая дата |
| `{{CURRENT_YEAR}}` | Текущий год |
| `{{BIRTH_DATE}}` | Дата рождения (из профиля пользователя) |
| `{{COURSE}}` | Курс обучения (из профиля или настройки Document:DefaultCourse) |
| `{{DIRECTION}}` | Направление подготовки (из профиля или Document:DefaultDirection) |
| `{{ACADEMY_NAME}}` | Название организации (из appsettings Document:AcademyName) |
| `{{STUDY_DURATION}}` | Срок обучения (из appsettings Document:DefaultStudyDuration) |
| `{{ENROLLMENT_ORDER}}` | Номер приказа о зачислении |
| `{{ENROLLMENT_ORDER_DATE}}` | Дата приказа о зачислении |
| `{{STUDY_START_DATE}}` | Плановый период обучения с |
| `{{STUDY_END_DATE}}` | Плановый период обучения по |

При генерации справки в шаблон автоматически подставляются данные пользователя и настройки из `appsettings.json` (секция Document). Подстановка работает даже если Word разбил плейсхолдер на несколько фрагментов.

## Пример текста справки об обучении (вставьте в .docx и замените подчёркивания на плейсхолдеры)

Скопируйте в Word и вместо «ФИО», «00.00.0000» и т.д. вставьте плейсхолдеры из таблицы выше (например `{{FULL_NAME}}`, `{{BIRTH_DATE}}`, `{{ACADEMY_NAME}}`):

```
Дана {{FULL_NAME}}, {{BIRTH_DATE}} года рождения в том, что он(а) действительно является студентом {{ACADEMY_NAME}}, обучается на {{COURSE}} курсе очной формы обучения на договорной основе без выплаты стипендии. Направление подготовки: {{DIRECTION}}. Плановый срок обучения в колледже {{STUDY_DURATION}}. Приказ о зачислении № {{ENROLLMENT_ORDER}}-УВР от {{ENROLLMENT_ORDER_DATE}} г. Плановый период обучения с {{STUDY_START_DATE}} г. по {{STUDY_END_DATE}} г.

         Заместитель директора
         по учебно-методической работе                                                    __________________/Л. И. К. Гасанова
```

**Важно:** в самом файле шаблона (.docx) должны стоять именно плейсхолдеры `{{FULL_NAME}}`, `{{BIRTH_DATE}}` и т.д., а не слова «ФИО» или «00.00.0000». Тогда при создании заявки тип «Справка об обучении» они подставятся автоматически.

Данные «Дата рождения», «Курс», «Направление подготовки» берутся из профиля пользователя (админ может указать их при редактировании пользователя). Название академии и срок обучения — из `appsettings.json`, секция `Document`.
